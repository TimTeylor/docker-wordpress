# Getting started

### Example .env file

```
# Wordpress
WORDPRESS_DB_HOST = "db"
WORDPRESS_DB_USER = "testuser"
WORDPRESS_DB_PASSWORD = "testpassword"
WORDPRESS_DB_NAME = "testdb"

# Mysql
MYSQL_DATABASE = "testdb"
MYSQL_USER = "testuser"
MYSQL_PASSWORD = "testpassword"
MYSQL_RANDOM_ROOT_PASSWORD = "1"
```
